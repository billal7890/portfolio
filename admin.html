Security and Deployment Notes

Static demo security

This Admin editor is protected for local/demo use only. Client-side passwords can be viewed by technical users after deployment, so do not treat this as production authentication.

Before publishing a real editable Admin online, connect authentication through Firebase Auth, Supabase Auth, Auth0, or a server-side login. Keep private editor actions behind backend rules, not only JavaScript.

  • Visitor pages stay clean and do not show edit controls.
  • admin.html includes noindex, nofollow to discourage search indexing.
  • Files uploaded in this static demo are stored in browser localStorage unless they are already part of the repository assets.
