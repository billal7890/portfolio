# Billal Javed Portfolio

Personal portfolio website focused on business analytics, data visualization,
simulation, finance, and information management.

## Run locally

From this folder, run:

```powershell
python -m http.server 8000
```

Then open:

```text
http://127.0.0.1:8000/
```

The site is built with HTML, CSS, and JavaScript. Static demo content and admin
changes are stored in the browser's localStorage.

## GitHub Pages

This repository can be deployed directly with GitHub Pages:

1. Open the repository's **Settings**.
2. Select **Pages**.
3. Under **Build and deployment**, choose **Deploy from a branch**.
4. Select the `main` branch and `/ (root)` folder.
5. Save and wait for the published URL.

The contact form uses FormSubmit and may require a one-time email activation.
